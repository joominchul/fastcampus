General Purpose Language
- 범용 목적의 언어
- 코틀린, 파이썬, 자바, C, C++, 스위프트
Domain Specific Language(DSL)
- 도메인 특화 언어 -> 특수 목적 달성을 위한 언어
- 문법 간단.
- XML(eXtensible Markup language)
    - 안드로이드 UI용 언어.
    - extensible
        - 태그 안에 태그 추가 가능.
        - 태그
            - <태그></태그>
            - <태그/>
    - Markup language
        - 태그로 마크를 하여 내용을 작성
